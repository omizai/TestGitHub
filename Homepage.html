Create a new project to GitHub

1. [Git hub] Create a repository

2. [Local Git] git init

3. [Local Git] git add .

4. [Local Git] git commit -m "inital commit"

5. [Local Git] git remote add origin https://github.com/omizai/xxx.git

Telling your local Git repository where to find this remote repository by adding it as a remote named origin

6. [Local Git] git push -u origin master

Create a branch on local

1. [Local Git] git checkout -b BranchName

stored a copied of code to a new branch

2. [Disk] Update code

3. [Local Git] git diff

Compare the diff between the branch and your current(updated) source code

4. [Local Git] git add .

Tell git you want to commit those files

5. [Local Git] git commit -m "message"

Commit change to Branch A at Local Git

6. [Local Git] git push origin BranchA

This will push(also create) BranchA to the github with BranchA source code

Merge 2 branch

1. [Local Git] git checkout master

Switch to master on local git(which is not get latest yet)

2. [Local Git] git pull origin master

Pull the source code from remote to update the master on local

3. [Local Git] git checkout BranchA

Switch to BranchA

4. [Local Git] git rebase master

Use master as base, Merge current branch code (BranchA) to it

5. [Local Git] git push -f origin BranchA

Push the rebased version back to github branchA

6. [Git hub] Create Pull request (pull BranchA to main)

7. [Git hub] Approve pull request. Use Squash and merge

8. [Git hub] Delete branchA on github

9. [Local Git] git checkout master

Switch to master before delete branch A

10. [Local Git] git branch -D BranchA
